Writing Blog Posts the Easy Way

Published June 17, 2026

I wanted to make it easier to write blog posts for this site. Instead of writing HTML, I can now just write plain text files.

Each text file contains metadata at the top (title, date, slug, and excerpt) followed by the content. The content is just paragraphs separated by blank lines.

A simple Python script converts these text files into HTML blog posts that match the site's style. It also automatically updates the blog index page.

Now I can focus on writing content instead of dealing with HTML tags and template boilerplate!
